# Fardin Saraf - Portfolio

My personal portfolio, live at [decea1st.dev](https://decea1st.dev).

It showcases who I am as a full-stack developer: my skills, the tech stack I work with, my experience, and the projects I've built, including [Clustr](https://clustrgoals.com/), [Outbreak.io](https://outbreakio.dev), and [CampusVoice](https://github.com/HashirAligb/campusvoice).

## Tech Stack

- **Framework:** Next.js (App Router) with React and TypeScript
- **Styling:** Tailwind CSS
- **Animation:** GSAP with ScrollTrigger
- **Deployment:** decea1st.dev

## Running Locally

```bash
pnpm install
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.
